<h2>Install Active Directory Domain Services (AD DS)</h2>
<br />

- **Open Server Manager**
- **Click Add Roles and Features**
- **Select:**

- **Choose server**
- **Select:**
    - **Active Directory Domain Services**
- **Install and complete wizard**
