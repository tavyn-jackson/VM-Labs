<h2>Add Active Directory roles and features:</h2>
<br />
- <b>Power on PC/Desktop</b>
<br />
- <b>Login into PC/Desktop using administrator account username and password</b>
<br />
- <b>After logging into admininstrator account, select "Manage"</b>
<br />
- <b>Select "Add Roles and Features", roles and features selection window will prompt</b>
<br />
<br />
<p align="center">
  <img src="Images/AddRoles.png" height="50%" width="70%" alt="Add Roles"/>
</p>
<br />
- <b>Select "Active Directory Domain Services"</b>
<br />
- <b>Select correct configurations as prompted</b>
<br />
- <b>After configurations select install which will require a reboot after installing</b>
<br />
<br />
<p align="center">
<img src="Images/InstallRoles.png" height="50%" width="70%" alt="Install Roles"/>
</p>
<br />
- <b>After reboot, will return back to the PC/Desktop OS login screen</b>
<br />
- <b>Login using administrator username and password </b>
<br />
- <b>After logging in the Server Manager will open again</b>
<br />
- <b>Select Tools in Server Manager, Active Directory will now be listed as an option</b>
<br />
